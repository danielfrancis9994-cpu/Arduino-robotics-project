# Ultrasonic Servo System

This project uses an ultrasonic sensor and servo motor to detect nearby objects and respond automatically.

## Components
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- Servo Motor
- Jumper Wires

## Features
- Distance detection
- Servo response movement
- Real-time sensing

## Future Improvements
- Faster response speed
- Obstacle tracking
- Buzzer alert system
